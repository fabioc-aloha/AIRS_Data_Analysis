# Results Directory

This directory contains analysis outputs from the AIRS psychometric validation study.

## Structure

```
results/
├── models/      # SEM model objects and saved analyses
├── plots/       # Visualizations and figures
└── tables/      # Statistical output tables
```

## Contents

### Tables (Version Controlled)
Small CSV files with statistical results are tracked for reproducibility:
- `efa_loadings.csv` - Exploratory factor analysis loadings
- `reliability_analysis.csv` - Cronbach's α, CR, AVE values

### Models (Not Tracked)
Large binary model objects are excluded from git:
- SEM model objects (`.pkl`)
- R model objects (`.rds`, `.RData`)

### Plots (Not Tracked)
Generated visualizations are excluded (regenerate from notebook):
- Factor loading plots
- Path diagrams
- Correlation heatmaps
- Distribution plots

## Regenerating Results

All results can be regenerated by running:

```bash
# Python analysis
jupyter notebook AIRS_Analysis_Python.ipynb

# R analysis (if applicable)
Rscript -e "rmarkdown::render('AIRS_Analysis.Rmd')"
```

### Phase 3: Measurement Invariance (Complete - November 23, 2025)
- **Configural Invariance**: SUPPORTED across Role (CFI=0.945), Usage (CFI=0.922), Adoption (CFI=0.946)
- **Metric Invariance**: NOT SUPPORTED (max loading differences: 0.414-0.481 exceed threshold)

- **Interpretation**: Constructs function differently across contexts (theoretically meaningful)
- **H4 Implications**: Use separate-group structural models for moderation analysis
